Always getting better.

Missing e is constantly improving! When there are changes to the Tumblr interface or when a great new feature idea comes up, you'll have the extension that is first to improve your Tumbling!

Completely customizable.

Whether you want to reply to replies, see all of a post's tags or reblog without leaving the Tumblr dashboard, Missing e has what you need! Something you don't want? Easy. You can disable any feature and customize all sorts of options!

Simpler and faster.

Some extensions for Tumblr can take forever to load. Even when they do, many don't integrate well with Tumblr's appearance. Do away with trouble like that by installing Missing e.

Safe, free and truly open.

Not only is Missing e provided entirely free of charge, it is also completely open source. That means anybody can examine the code to confirm that it is safe. Don't settle for an extension whose code you can't check out for yourself! Source Code on Github
