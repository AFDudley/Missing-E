Always getting better.

Missing e is constantly improving! Whether there are changes to the Tumblr interface to be improved or a great new feature idea, you'll be using an extension that strives to always be fresh and exciting!

Stay updated on Tumblr

Completely customizable.

Whether you want to reply to replies, see all of a post's tags or reblog without leaving the Tumblr dashboard, Missing e has what you need! Something you don't want? Easy. You can disable any feature and customize all sorts of options!

Safe, free and open.

Not only is Missing e provided entirely free of charge, it is also completely open source. That means anybody can examine the code to confirm that it is safe and anybody can help make the extension better!
